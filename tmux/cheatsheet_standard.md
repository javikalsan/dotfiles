# tmux

## session

* attach `tmux a`
* attach to named `tmux a -t name`


## windows

*create new* `CTRL + c`
*go to window number* `CTRL + b + number`
*move window* `CTRL + b + .` and place the tab number destination


## panes

*vertical split* `CTRL + b + %`
*horizontal split* `CTRL + b + "`
*go to next pane* `CTRL + b + o`
